Simbolių eilutės

Darbui su simbolių eilutėmis yra sukurta klasė(tipas) string. Failas, kuriame yra aprašyta klasė string, į programą įterpiamas sakiniu: #include <string> .

Simbolių eilučių tipo kintamieji (objektai) aprašomi taip pat, kaip ir kitų tipų kintamieji: string eil, e1.

Sukurtos eilutės yra tuščios. Jas aprašant galima priskirti reikšmes: string eil = Jau moku programuoti C++ kalba

Priskyrimo:
string s1, s2;
string s = 12;
char c = 'z';
s1 = s; s2 = c;

Rezultatas – s1 reikšmė (12) ir s2 reikšmė (z).
