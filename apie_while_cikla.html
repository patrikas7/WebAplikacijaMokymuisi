Ciklas "while"

Ciklas while yra naudojamas, kai norime vykdyti tam tikrą kodą tol, kol tenkinama sąlyga.

While ciklo sintaksė:

int i = 0;
while(i < 10) {
  kodas kurį norime kartoti
  i++;
}

Šiuo atveju yra labai svarbu nepamiršti padidinti kintamojo i, esančio while ciklo kamiene. Jei to nepadarysime, bus užsuktas begalinis ciklas. Dar vienas pavyzdys while ciklui:

Tarkime, jog norime sužinoti, kiek kartų skaičius 1024 dalinasi iš dviejų. Tam nustatyti galėtume pabandyti 1024 dalinti iš dviejų iki tol, kol daugiau padalinti be liekanos nebeišeis. Kodas atrodytų taip:

int x = 1024;
int kiek = 0;
while (x % 2 == 0) {  /* kartosime tol, kol x dalinsis iš dviejų be liekanos*/
  x /= 2;  // padaliname x iš dviejų
  kiek++;  // padidiname atsakymą vienetu
}
cout << kiek << endl; // atspausdiname atsakymą

Šis kodas išvestų skaičių 10.
