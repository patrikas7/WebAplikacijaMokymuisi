Išrinkimo sakinys "switch"

Išrinkimo sakinys gautą išraišką (kintamasis skliaustuose) tikrina su konstanta, kuri aprašoma case komanda, ir jei išraiška yra lygi konstantai, vykdomi jos sakiniai (sakinys), jei ne - vykdomi default žymės sakiniai. Pastarosios gali ir nebūti. Kiekvienoje konstantoje patartina naudoti komandą break, taip išvengiama painiavos vykdyme. Išraiška gali būti tik sveiko ar simbolinio tipo. Tai reiškia, kad switch galima naudoti tik su int, char arba string tipais.

Switch išrinkimo sakinio pavyzdys:

int a;
cin >> a;
switch(a){
  case 1: // jūsų kodas
  break;
  case 2: // jūsų kodas
  break;
  case 3: // jūsų kodas
  break;
}

Šiuo atveju priklausomai nuo į konsolę įvesto skaičiaus, atliekami atm tikri veiksmai.
