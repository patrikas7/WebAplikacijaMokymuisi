Sąlyginis sakinys "if"

Šnekamojoje kalboje dažnai sakome “Jei bus X, tai darysiu Y”. Pavyzdžiui, “Jei šiandien lis, tai būsiu namie, o kitu atveju eisiu žaisti į lauką”. Lygiai tokias pačias sąlygas dažnai prisireikia nurodyti ir programuojant kompiuteriui ir tam naudojami sąlyginiai sakiniai. Sąlyginis sakinys - tai kodo dalis, nurodanti, kokioms sąlygoms esant reiktų vykdyti (ar nevykdyti) tam tikrą kodą.

If sąlyginio sakinio užrašas atrodo taip:

if(i < 10){
  // kodas, kurį norime vykdyti jei sąlyga yra teisinga
}

Šiuo atveju bus tikrinama ar kintamasis pavadinimu i yra mažesnis už 10, jei tai tiesa, bus vykdomas kodas if sakinio kamiene, jei bus netiesa, kodas bus nevykdomas. Jeigu norime padayti, kad būtų vykdomas tam tikras kodas neatitikus sąlygai, turime naudoti žodelį else.

if(i < 10){
  // kodas, kurį norime vykdyti jei sąlyga yra teisinga
}
else {
  // kodas, kurį norime vykdyti, jei sąlyga neteisinga
}

Šiuo atveju jei i bus didesnis už 10, tai bus nueinama į else bloko dalį ir bus vykdomas kodas jo kamiene. Taip pat galime ir sujungti šiuos abu žodelius jei norime daugiau nei dviejų sąlygų:

if(i < 10) {
  // kodas, kurį norime vykdyti jei sąlyga yra teisinga
}
else if (i < 20){
  // kodas, kurį norime vykdyti, jei sąlyga teisinga
}
else {
  // kodas, jei nei viena sąlyga netenkinama
}
